Normal input and textarea

Create normal input
only for normal input with type="text", type="email", type="number", type="password" and textarea fields
Note : type="radio", type="checkbox" or select dropdown has different layout for custom design
class="form-input-optional" should be given when the input doesn't requires validation
Threfore it has no response div associated with it you will understand it more clearly in next block

Normal input

Create input with icon
class="form-input" will be used by script to validate any normal input and textarea field
class="form-input-response" will be used to print the validation response

Radio inputs

Create custom radio input with or without icon
If you want these radio to be required just add the name attribute value to script here name="gender" so put 'gender' in array

Checkbox inputs

Create custom checkbox input with or without icon
If you want these radio to be required just add the name attribute value to script here name="company[]" so put 'company[]' in array

Select dropdown

Create custom select dropdown with or without icon
If you want the select dropdown to be required just add the name attribute value to script here name="language" so put 'language' in array
